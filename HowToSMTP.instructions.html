To set the SMTP server directly in the script itself, you'll need to use a library like PHPMailer or SwiftMailer because PHP's native mail() function does not support SMTP settings (like authentication, encryption, etc.) directly in the script.

Hereâ€™s how you can do this using PHPMailer, which is one of the most popular libraries for sending emails via SMTP in PHP.

Steps to Send Mail Using PHPMailer (with SMTP)

  1. Download and Install PHPMailer

    You can install PHPMailer using Composer or download it manually from GitHub.

    • Via Composer (if you are using Composer):

      bash
      composer require phpmailer/phpmailer
    • Manually: Download PHPMailer from PHPMailer GitHub and include it in your project.



  2. Use PHPMailer in Your Script

    Here is an example of how to configure SMTP settings directly in a PHP script using PHPMailer.

    php
    <?php
// Load PHPMailer classes (assuming you're using Composer)
use PHPMailer\PHPMailer\PHPMailer;
use PHPMailer\PHPMailer\Exception;

require 'vendor/autoload.php';  // Adjust this path if you've installed PHPMailer manually

// Create a new PHPMailer instance
$mail = new PHPMailer(true);

try {
    //Server settings
    $mail->isSMTP();                                    // Set mailer to use SMTP
    $mail->Host       = 'smtp.yourserver.com';           // Specify main SMTP server
    $mail->SMTPAuth   = true;                            // Enable SMTP authentication
    $mail->Username   = 'your_username@yourserver.com';  // SMTP username
    $mail->Password   = 'your_password';                 // SMTP password
    $mail->SMTPSecure = PHPMailer::ENCRYPTION_STARTTLS;  // Enable TLS encryption; use PHPMailer::ENCRYPTION_SMTPS for SSL
    $mail->Port       = 587;                             // TCP port to connect to (e.g., 587 for TLS, 465 for SSL)

    //Recipients
    $mail->setFrom('your_username@yourserver.com', 'Mailer');
    $mail->addAddress('recipient@example.com', 'Joe User');  // Add a recipient

    // Content
    $mail->isHTML(true);                                // Set email format to HTML
    $mail->Subject = 'Here is the subject';
    $mail->Body    = 'This is the HTML message body <b>in bold!</b>';
    $mail->AltBody = 'This is the body in plain text for non-HTML mail clients';

    // Send the email
    $mail->send();
    echo 'Message has been sent';
} catch (Exception $e) {
    echo "Message could not be sent. Mailer Error: {$mail->ErrorInfo}";
}
?>

Explanation of Key Parts:

  • $mail->isSMTP();: Tells PHPMailer to use SMTP.
  • $mail->Host: The address of your SMTP server.
  • $mail->SMTPAuth: Enables SMTP authentication.
  • $mail->Username and $mail->Password: Your SMTP username and password.
  • $mail->SMTPSecure: Encryption protocol (TLS or SSL).
  • $mail->Port: The SMTP port, usually 587 for TLS and 465 for SSL.
  • setFrom, addAddress, Subject, and Body: Defines the sender, recipient, subject, and body of the email.

PHPMailer Features:

  • SMTP Authentication: PHPMailer supports username/password authentication for SMTP.
  • Encryption: It supports both TLS and SSL.
  • Error Handling: PHPMailer provides robust error handling.

This method provides much more flexibility and is generally more secure and reliable than PHP's mail() function.
